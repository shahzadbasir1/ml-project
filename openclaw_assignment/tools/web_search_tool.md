# Web Search Tool

## Purpose

Provides access to online developer documentation and reference materials.

## Capabilities

* Search developer portals
* Retrieve API documentation
* Discover authentication methods
* Gather implementation examples

## Inputs

* URL
* Search query

## Outputs

* Documentation summary
* API references
* Authentication information

## Example

Input:

https://developer.shopify.com

Output:

Developer portal summary and API catalog
