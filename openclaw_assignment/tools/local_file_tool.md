# Local File Tool

## Purpose

Provides access to local files for analysis and documentation generation.

## Capabilities

* Read Postman collections
* Read OpenAPI specifications
* Read text files
* Save generated reports

## Inputs

* File path
* API specification file
* Postman collection

## Outputs

* Parsed content
* Structured API information

## Example

Input:

shopify_collection.json

Output:

List of APIs and request examples
