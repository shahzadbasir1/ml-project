# API Analysis Agent

## Purpose

The API Analysis Agent analyzes API specifications and developer documentation.

## Responsibilities

* Discover APIs
* Analyze authentication methods
* Extract endpoints
* Identify request and response structures

## Inputs

* Developer portal URLs
* OpenAPI specifications
* Postman collections

## Outputs

* API inventory
* Authentication summary
* Endpoint documentation

## Example

Input:

Shopify Developer Portal

Output:

* Authentication Method
* Available APIs
* Endpoint List
* Request Parameters
