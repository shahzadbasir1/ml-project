# Planning Agent

## Purpose

The Planning Agent is responsible for understanding user requests and determining the workflow required to fulfill the request.

## Responsibilities

* Analyze user intent
* Determine which agents are required
* Route tasks to appropriate agents
* Coordinate workflow execution

## Inputs

* User requests
* Task descriptions

## Outputs

* Workflow plan
* Agent routing decisions

## Example

User Request:

"Analyze Shopify APIs and create a testing plan."

Workflow:

1. Route to API Analysis Agent
2. Route results to Documentation Agent
3. Generate final report
