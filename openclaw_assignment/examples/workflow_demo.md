# Workflow Demonstration

## Use Case

Integration Analyst Assistant

## User Request

Research Shopify APIs and provide an overview of:

1. Authentication methods
2. Product APIs
3. Order APIs
4. Customer APIs

## Workflow

Step 1:
User submits request through OpenClaw Chat.

Step 2:
GPT-5.5 receives the request.

Step 3:
OpenClaw invokes the Web Search tool.

Step 4:
Shopify documentation is researched.

Step 5:
Assistant generates a structured API overview.

## Result

Generated:

* Authentication overview
* Product API summary
* Order API summary
* Customer API summary
