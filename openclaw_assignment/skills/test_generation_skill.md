# Test Generation Skill

## Purpose

The Test Generation Skill automatically generates positive, negative, and edge test scenarios for API endpoints.

## When To Use

Use this skill whenever the user wants to:

* Generate API test cases
* Create QA scenarios
* Validate endpoint behavior

## Inputs

* API Endpoint
* Request Schema
* Response Schema

## Outputs

* Positive Test Cases
* Negative Test Cases
* Boundary Test Cases
* Error Scenarios

## Example

Endpoint:

GET /products

Output:

* Valid Request Test
* Invalid Token Test
* Missing Parameter Test
* Rate Limit Test
* Empty Result Test
