# Documentation Skill

## Purpose

The Documentation Skill generates technical documentation for APIs and integrations.

## When To Use

Use this skill whenever the user wants:

* Integration Guides
* Business Rules
* Error Handling Documentation
* Technical Summaries

## Inputs

* API Analysis Results
* Sample Requests
* Sample Responses

## Outputs

* Integration Guide
* Business Rules
* Error Handling Section
* API Documentation

## Example

Input:

Authentication Details
Request/Response Samples

Output:

Complete Integration Guide
