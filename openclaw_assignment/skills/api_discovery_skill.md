# API Discovery Skill

## Purpose

The API Discovery Skill is responsible for discovering and cataloging APIs from developer portals, OpenAPI specifications, Swagger documentation, and Postman collections.

## When To Use

Use this skill whenever the user wants to:

* Analyze a developer portal
* Discover available APIs
* Understand authentication requirements
* Extract endpoint information

## Inputs

* Developer Portal URL
* OpenAPI Specification
* Swagger Documentation
* Postman Collection

## Outputs

* Authentication Summary
* Endpoint Inventory
* Request Parameters
* Response Structures

## Example

Input:

https://developer.shopify.com

Output:

* OAuth Authentication
* Product APIs
* Order APIs
* Customer APIs
